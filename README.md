## Pinout
| ESP32 | STM32_M7   | Function   |
|-------|------------|------------|
| I019  | PB14 (D12) | MISO       |
| I018  | PA12 (D13) | CLK        |
| IO23  | PB15 (D11) | MOSI       |
| IO5   | PA11 (D10) | CS         |
| IO2   | PJ1 (D2)   | HS         |
| IO4   | PJ3 (D7)   | DATA READY |
| EN    | PJ0 (D4)   | RESET ESP  |
| GND   | GND        | Ground	  |
